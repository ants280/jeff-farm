• Once logged in Users can update their information and change their password on the Users page.
  • Farms
    • Once created, farms can be shared with other users on the Permissions page. Beware that any user with access to a farm can grant/revoke others' access to the farm.
    • Hives and Poultry records can be created from the Farms page.
  • Hives/Poultry
    • Hives/Poultry records can be created from the Farms page.
    • Individual hive/poultry characteristics can be set for each record.
    • The Inspections grouping displays the recent inspections of each Hive/Poultry record.
  • Inspections
    • Inspections can be made for one or more target (Hive/Poultry record). This allows tracking of certain attributes between inspections.
    • The Charts page is available for inspections to visually see how numerical inputs change between inspections.
